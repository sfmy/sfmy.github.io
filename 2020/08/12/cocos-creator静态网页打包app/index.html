借助cordova打包静态网页(mac环境)

  1. 下载安装nodejs与npm
    brew install nodejs 安装nodejs
    npm速度比较慢的话可以更换淘宝镜像或者安装cnpm
    npm install -g cnpm
  2. 安装cordova
    cnpm install -g cordova
  3. cordova创建测试项目
    cordova create demo com.test.demo Demo
  4. cordova运行测试
    cordova platform add android
    cordova serve
