hexo-github搭建

发表于 2024-08-30 分类于 技术
1.新建发布仓库

仓库名字格式为<用户名>.github.io
此仓库用来发布内容

2.新建博客仓库

新建仓库blogs
此仓库用来存储博客内容方便不同电脑同步编辑内容

3.本地部署hexo环境

拉取远程github.io仓库到本地
新建blogs文件夹
部署hexo环境
把source文件夹内容与blogs对应

4.发布博客到github.io

在blogs文件夹中写自己的博客
hexo g 生成静态文件
把public中的文件全部拷贝到github.io中并提交
即可在github.io中访问自己的博客

5.修改配置文件

修改本地的config.yml文件和主题中的config.yml文件
打开tags和categories设置

6.本地添加自定义脚本

自动添加tag和categories
自动发布github.io
