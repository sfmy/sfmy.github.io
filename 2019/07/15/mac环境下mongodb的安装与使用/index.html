sfmy
2019-07-15
tools
  1. 使用homebrew安装
    brew install mongodb
  2. 安装完成后启动mongodb
    mongod –config /usr/local/etc/mongod.conf
  3. 或者通过 brew services start mongodb启动
    brew services stop mongodb关闭
