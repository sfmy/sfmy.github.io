游戏数据驱动界面思考

发表于 2024-05-22 更新于 2024-08-29

游戏前端，本质是一个画面的呈现。前端技术的提高围绕着画面呈现的更完美，更炫酷，体验更丝滑，性能更好这
方面。前端的显示，是数据的显示。如果是网络类型的游戏，在界面展示时再请求数据就会导致界面的刷新，而有
不好的体验。这个时候，把数据和页面分离出来，就好多了。页面销毁，而数据不销毁，或者页面和数据都不销毁。
再次打开页面，也就丝滑顺畅了许多。缺点是，不销毁页面占用内存。

永生的思考

发表于 2024-05-08 更新于 2024-08-30 分类于 技术

人的一生很像是一场游戏，只有一条命，死亡也就游戏结束了。不过每个人开场的属性不一样，基于此再加上无数
个的生命中的选择，造成了多彩的世界。

而永生，一直是人类的追求。可以预测的是，永生早晚会实现。不过此实现是福是祸对于人类来讲就很难确定了。
譬如游戏外挂，虽然降低了游戏的难度，但是也导致游戏变得索然无味。

一个人的记忆，决定了这个人是谁。简单来说就是，一个人失忆了，那么此前这个人的一切也就消失了。剩下的只
是躯壳而已。这有点像玄幻小说中的夺舍一样，一个人被夺舍了，他也就消失于天地间了。同样，夺舍的人，可以
凭此躯壳继续生活下去，带着原来的记忆。

而永生，就是夺舍。只不过夺舍的对象不是人类，而是其他符合法律法规的东西。凭此让一个人的记忆思想存续下
去。这些在医学发展到一定的阶段就可以实现了。当然，或许医学发展到一定的条件，躯壳也可以长生不老，这样
也可以在不更换躯壳的情况下直接实现永生。不过如果记忆可以转移，或许有人可能尝试当一棵树，当一条狗，也
会是比较有意思的事情。只要把记忆转移到树或狗身上，就是一个另类的体验。

只不过，真有这些技术，此时也差不多类似游戏通关了。生命也就无意义了吧。

记忆可以转移，存储，也就可以复制修改保存。每个人也就具化成一堆数据属性，仅此而已，不知道这样还算不算
是一条生命。

研究的这么透彻，有了这么多的属性，当然构建一个虚拟世界也不成问题。所以我们生活的世界的真实性，也无从
考据。正如人类技术的发展发现，指不定是服务器的升级，能承载更多的算力，允许人类的发展与发现一样。

换皮二开游戏公司的套路

发表于 2024-05-07 更新于 2024-08-29

最近找工作，遇到很多换皮游戏公司。换皮游戏公司能生存下来并做大，一定有它存活的方式。

游戏公司的客户，相对专业的游戏制作人，都很业余。大部分应该都是听说游戏行业比较赚钱，想参与进来试试。
但是很多专业的游戏公司，都在赔钱开发游戏，就更不要说这些业余的人，他们肯定更难赚到钱。除非他们有一
些野路子，跟普通的游戏赚钱模式不一样。私服是一个路子，这个只要游戏做的不是太差，也有一批人愿意往里
面充钱。刷广告也是一个路子，靠培养权重，提高看广告的价值，看自己游戏里的广告盈利。再有一些路子，就
是棋牌类型了，涉及房卡与赌博的。这些都不靠游戏质量赚钱，不涉及与其他游戏进行竞争。所以二开与换皮也
就无所谓了。

那么换皮游戏公司是什么套路呢，他们怎么盈利的？自己开发，或者买一套源码，重复利用。给客户自己选游戏，
或者客户制定完游戏，再从其他渠道收购源码。简单改改美术表现，就算是交付了。这其中，估计没有多少游戏
公司会在意游戏的质量，有的游戏体量太大，公司可能就没有奔着交付去努力。收一个前期与中期的款项就不在
乎尾款了。客户催的话，就提要求加钱和加时间。不过，钱和时间也解决不了问题，再继续投也不过是陷的更深。
当然，遇到一些比较简单的项目，换皮也有可能做到最后可以正常交付使用。概率较小而已。

我所见过的换皮公司，基本都是在坑蒙拐骗。当然，跟客户打官司少不了。打官司这个太耗时间和精力，一般人
估计也就认栽了。换皮公司对游戏从业者也是极其不友好的，因为换皮公司的存在，霸占了正常的游戏开发岗位，
而且没有什么技术上升空间。奉劝各位，能少接触换皮和二开还是少接触吧。

世界发展是在进步还是倒退

发表于 2023-12-12 更新于 2024-08-29

我是一个90后，从小在农村长大。小学与初中都是在村里上的学，高中是在县城上的学，大学是在一线城市上的学。
我觉得我算是处在时代过渡中的中间层，能理解上一代与下一代的思想，同时也看了城市的兴旺与农村的凋零，看
到了人们生活方式的变化。

在我看来，父辈们已经算是摆脱了温饱的忧愁，同时没有物质上太多的追求，以及来自工作的压榨。他们可支配的
自由时间充足，算是比较幸福的一代人。而我们这一代，被迫由农村转向城市生活，这个转换进度有点太快，导致
这代人生活的很累。农村生活，条件简朴，消费低，节奏慢，一切东西都可从土地获得，可持续性强。城市生活则
相反，条件优越，消费高，节奏快，所有的生活活动都需要金钱的支撑。而如果一个社会并没有能保障一个人的工
作稳定与可持续，城市生活有很大的隐患。

社会的进步看生产力是否提高了，但是我对此却不太认同。我看现在的情况是物质进步了，但是精神层面的东西却
丢失了。过去一个人日产一千，能够满足衣食住行，现在一个人日产一万，能够满足衣食住行，外加奢侈品。过去
实际工作可能不到5小时每天，现在感觉实际工作9小时每天。虽然物产丰富了，但是自由可支配的时间少了。社会
进步的终极目标，应该是要把人从繁重的劳动中解脱出来，过上自由自在的生活，摆脱物质层面的束缚，按照自己
喜欢的方式生活。如果这么一想，虽然积累了财富，但是人的本性却压抑的更狠了，这也就不奇怪，为什么以前人
脸上笑的更开心了。

农村是否是被抛弃了，我觉得这可能是一个阴谋，也可能是自然而然的走到了这一步。我对农村是怀有一种说不上
来的感情的，有可能是我从小生活在那里，家在那里，这很难说清楚。总之，当我看到麦田玉米地，总会感觉熟悉
开心。上一辈或许还有惧怕的心里，又爱又恨，因为他们被田地累坏过。我却是从小在田里玩着长大的，并没有经
历过这种苦。以前，农村人没有见过城市里的种种，也没有在城市里生活过，再加上农村的清苦与城市的繁华对比，
所以城市可能有莫名的吸引力。再加上望子成龙思想，在外谋个营生比守这一片产出不高的土地强太多。不过，出
去太远，家就荒了。如果城市扁平化发展，那么大部分人可以就近谋生。但是城市掠夺式发展，吸了周边太多的资
源，又没反哺周边的话，就导致很多人在附近很难谋生。只能去大城市寻出路，精壮劳动力和消费全留在了大城市。
农村只剩一些儿童老人，这样破败是迟早的问题。

我不知道有没有意识到这个问题，或者也无力解决这种问题。城市吸纳年轻人，同时得为他们提供后续的衣食住行
的保障。不然，就是像嚼甘蔗一样，榨干了汁液，再吐掉残渣，残忍之极。而农村却是一个小的循环，资源在土地
和家中循环流转。如果不人为干涉，自成一个体系。出生，长大，结婚生子，都在小范围的圈子里生活。生产不多，
消耗不多，没有大富大贵的想法，可以悠闲自在一生。

不同年龄有不同的看法吧。
